## Repository Structure

Contains fundamental SQL queries such as SELECT, WHERE, GROUP BY, ORDER BY, and basic filtering.

Intermediate_SQL
Includes joins, subqueries, Common Table Expressions (CTE), and window functions.

Advanced_SQL
Covers performance tuning, indexing, and complex query patterns used in Data Engineering.

Sample datasets used for query practice.
